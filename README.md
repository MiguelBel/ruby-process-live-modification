# Ruby process live modification

Some snippets for modifying the behavior of a ruby process. With/without having access to the process and/or with/without modifying the source code.

Basically is the usage of [rbtrace](https://github.com/tmm1/rbtrace) and [eval](https://www.rubydoc.info/stdlib/core/Kernel:eval).
